# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Проект посвящён первичному анализу данных (EDA) и оценке их качества. 
В рамках работы выполняется:
- Загрузка и изучение структуры датасета
- Проверка на пропуски, дубликаты и выбросы
- Визуализация распределений и зависимостей
- Составление отчёта о качестве данных

## Структура репозитория

```
data/ # Исходные данные
notebooks/ # Jupyter Notebook с анализом
report/ # Текстовый отчёт
.gitignore # Игнорируемые файлы
README.md # Описание проекта
```
## Используемые данные
**Датасет:** TV Shows Dataset  
**Источник:** открытые данные о телевизионных шоу  
**Размер:** 5611 строк, 10 столбцов  
**Описание признаков:**
- `Title` - название шоу
- `Year` - год выпуска
- `Age` - возрастной рейтинг (7+, 16+, 18+, all)
- `IMDb` - рейтинг на IMDb (от 1 до 10)
- `Rotten Tomatoes` - рейтинг на Rotten Tomatoes (в процентах)
- `Netflix`, `Hulu`, `Prime Video`, `Disney+` - доступность на платформах (1 - есть, 0 - нет)
- `type` - служебный признак
